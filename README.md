# ReturnZero-ZRHS2017
ZeroRobotics code for 2017 High School Tournament

Questa è la repository che contiene tutto il codice del team ReturnZero che riguarda il campionato internazionale ZeroRobotics per l'anno 2017.
Il proprietario della repo in questione sono io, Mimmo; tra l'altro, sono anche il vostro team mentor.
La repo è privata, quindi anche se è indicizzata da Google, i contributor non autorizzati non potranno visualizzare/modificare il contenuto.
Vi chiedo, per mantenere la repository ordinata, di committare e pushare consapevolmente nelle branch adeguate.
È vero, si può tornare indietro con un clic, ma ciò non significa che ogni volta io debba mettermi a pulire un casino di roba.
